Around the Web

YouTube Timestamp Search

YouTube Timestamp Search enables users to search for clips of videos rather than entire videos. It leverages YouTube API to fetch videos and transcripts, and OpenAI API's GPT model to generate embeddings to calculate similarity.

Made using Flask, HTML and CSS.

Fast Travel

Fast Travel is a one stop web app that allows users to quickly plan entire trips. It involves researching and scheduling attractions, flights and hotels into a printable ordered itinerary.

Made using React, Django and PostgreSQL.
Project Github

Sports For Kids

This is a web app made for the Texas A&M chapter of Sports For Kids. It handles essential functions such as membership tracking, information, event registration and updating profile. It also automates the application review process by allowing officers to accept application members directly into the roster.

Made using Ruby on Rails, HTML, TailwindCSS and PostgreSQL.
Project Github

Dis-Locate

Dis-Locate is a web app that was created at TAMUhack 2020. It allows you to enter the name of a disaster and it returns the top 5 locations where that disaster might be happening. It can also be used to predict where other emergencies such as shootings may be happening. It uses the Twitter API to scrape through the latest 10000 tweets related to the query and returns the locations.

Made using Python, HTML, CSS, Javascript
Project Github

Don't Panic

Don't Panic is an app that was created at HackTX 2019. It primarily aids those that are affected by natural disasters but it has endless applications in other emergency scenarios. On pressing the "Panic" button, Don't Panic sends your location and a voice message to emergency contacts chosen during sign up.

Made using PyQT, Google Maps API, Twilio API, gTTS module
