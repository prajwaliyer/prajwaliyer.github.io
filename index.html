Around the Web

Dis-Locate

Dis-Locate is a web app that was created at TAMUhack 2020. It allows you to enter the name of a disaster and it returns the top 5 locations where that disaster might be happening. It can also be used to predict where other emergencies such as shootings may be happening. It uses the Twitter API to scrape through the 10000 latest tweets related to the query and returns the locations. I worked on the front end of this project.

Made using Python, HTML, CSS, Javascript
https://github.com/prajwaliyer/TAMUHACKv2.0

Don't Panic

Don't Panic is an app that was created at HackTX 2019. It primarily aids those that are affected by natural disasters but it has endless applications in other emergency scenarios. On pressing the "Panic" button, Don't Panic sends your location and a voice message to emergency contacts chosen during signing up.

Made using PyQT, Google Maps API, Twilio API, gTTS module
